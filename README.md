# Fetal-Health-Classification
O projeto visa desenvolver um modelo preditivo para classificar o risco de complicações na gestação, utilizando dados de monitoramento fetal, como batimentos cardíacos e movimentos. O modelo classifica os fetos como saudáveis, em risco moderado ou em risco elevado, ajudando médicos a identificar problemas precoces e tomar decisões rápidas.
